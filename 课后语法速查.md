## 初级1 第1课

### 1. 名詞1は 名詞2です — 判断句（...是...）

- 结构：**名詞1＋は＋名詞2＋です**
- 「は」是提示助词，提示主题，读作「わ」。「です」表示判断，同时表示对听话人的礼貌。
- 例：わたし**は** マイク・ミラー**です**。（我是迈克·米勒。）
- 例：サントスさん**は** 会社員(かいしゃいん)**です**。（桑托斯是公司职员。）

### 2. 名詞1は 名詞2じゃ（では）ありません — 判断句否定（...不是...）

- 结构：**名詞1＋は＋名詞2＋じゃありません**
- 「じゃありません」是「です」的否定形。正式场合用「ではありません」。
- ⚠️ 口语用「じゃありません」，书面/正式用「ではありません」。
- 例：サントスさんは 学生(がくせい)**じゃありません**。（桑托斯不是学生。）

### 3. 名詞1は 名詞2ですか — 判断句疑问（...是...吗？）

- 结构：**名詞1＋は＋名詞2＋ですか**
- 句末加「か」构成疑问句，语调上升。
- 肯定回答：**はい、～です。** 否定回答：**いいえ、～じゃありません。**
- 例：ミラーさんは アメリカ人(じん)ですか。→ はい、アメリカ人です。
- 例：ミラーさんは 先生(せんせい)ですか。→ いいえ、先生じゃありません。会社員です。

### 4. 疑問詞＋ですか — 用疑问词提问

- 结构：**疑問詞＋ですか**
- 常用疑问词：どなた（哪位）、何歳(なんさい)（几岁）、おいくつ（贵庚）等。
- 回答时直接用具体内容替换疑问词，不用はい/いいえ。
- 例：あの 方(かた)は **どなた**ですか。→ ミラーさんです。

### 5. 名詞も — 也

- 结构：**名詞＋も**
- 「も」替换「は」，表示"也是"，对不同主题陈述相同内容。
- ⚠️ 用「も」时不能同时用「は」。
- 例：ミラーさんは 会社員です。グプタさん**も** 会社員です。（米勒是公司职员。古普塔也是公司职员。）

### 6. 名詞1の 名詞2 — 所属/属性

- 结构：**名詞1＋の＋名詞2**
- 「の」连接两个名词，名詞1修饰名詞2，表示所属、属性等。
- 例：ミラーさんは IMC**の** 社員(しゃいん)です。（米勒是IMC的职员。）
- 例：わたしは 大学(だいがく)**の** 先生です。（我是大学的老师。）

## 初级1 第2课

### 1. これ/それ/あれ — 指示代词（这个/那个/那个）

- 结构：**これ/それ/あれ＋は＋名詞＋です**
- 「これ」：离说话人近的东西。「それ」：离听话人近的东西。「あれ」：离双方都远的东西。
- 例：**これ**は 辞書(じしょ)ですか。→ いいえ、それは 辞書じゃありません。
- 例：**あれ**は 何(なん)ですか。→ あれは かばんです。

### 2. この/その/あの＋名詞 — 指示连体词

- 结构：**この/その/あの＋名詞**
- 修饰名词时不能用これ/それ/あれ，必须用この/その/あの。
- ⚠️ 「これ」单独使用，「この」后面必须接名词。
- 例：**この** 本(ほん)は わたしのです。（这本书是我的。）
- 例：**あの** 方(かた)は どなたですか。（那位是哪位？）

### 3. そうです/そうじゃありません — 肯定/否定确认

- 结构：**はい、そうです。/ いいえ、そうじゃありません（違(ちが)います）。**
- 用于回答「～ですか」的判断疑问句。否定时「違います」更常用。
- ⚠️ 只能用于名词判断句的回答，不能用于动词句。
- 例：それは 辞書ですか。→ はい、**そうです**。/ いいえ、**違います**。

### 4. ～か、～か — 选择疑问句

- 结构：**名詞1＋ですか、名詞2＋ですか**
- 从两个选项中选择，回答时直接说选择的内容，不用はい/いいえ。
- 例：これは「9」ですか、「7」ですか。→「9」です。

### 5. 名詞1の 名詞2 — 扩展用法（内容/产地/所有）

- 结构：**名詞1＋の＋名詞2**
- 第2课扩展：名詞1可以表示内容、产地、所有者等。
- 例：これは コンピューター**の** 本です。（这是关于电脑的书。→ 内容）
- 例：これは わたし**の** 本です。（这是我的书。→ 所有）

### 6.「の」代替名词用法

- 结构：**名詞1＋の**（省略后面的名词）
- 前文已出现的名词可以用「の」代替，避免重复。仅限物品，不能代替人。
- ⚠️ 不能用于代替人。
- 例：あれは だれの かばんですか。→ 佐藤(さとう)さん**の**です。（= 佐藤さんのかばんです）

## 初级1 第3课

### 1. ここ/そこ/あそこ/どこ — 场所指示代词

- 结构：**ここ/そこ/あそこ**（这里/那里/那里）
- 距离规则同これ/それ/あれ：ここ（说话人附近）、そこ（听话人附近）、あそこ（双方都远）。
- 礼貌形：こちら/そちら/あちら/どちら。
- 例：お手洗(てあら)いは **あそこ**です。（洗手间在那边。）

### 2. 名詞は 場所です — 说明所在地

- 结构：**名詞＋は＋場所＋です**
- 说明某人/某物/某场所在哪里。
- 例：電話(でんわ)は 2階(かい)です。（电话在2楼。）
- 例：山田(やまだ)さんは 事務所(じむしょ)です。（山田在办公室。）

### 3. ～は どこですか — 询问地点

- 结构：**名詞＋は＋どこですか**
- 询问某事物/场所的位置。礼貌形用「どちら」。
- 例：お手洗いは **どこ**ですか。→ あそこです。
- 例：エレベーターは **どちら**ですか。→ あちらです。

### 4. ～は ～の 名詞です — 产地/品牌/所属

- 结构：**名詞1＋は＋名詞2＋の＋名詞3＋です**
- 例：これは 日本(にほん)**の** カメラです。（这是日本的相机。）

### 5. どちら — 礼貌的"哪里"

- 「どちら」是「どこ」的礼貌形，也可用于询问国家/公司等。
- 例：会社(かいしゃ)は **どちら**ですか。→ IMCです。
- 例：お国(くに)は **どちら**ですか。→ アメリカです。

### 6. ～も ～です — 也是...

- 结构：**名詞1＋も＋名詞2＋です**
- 例：これは 500円です。あれ**も** 500円です。（这个500日元。那个也是500日元。）

### 7. ～は ～ですか、～ですか — 选择疑问

- 结构同第2课。用于场所/物品等的选择。
- 例：ワインは どちらですか、フランスですか、イタリアですか。→ イタリアです。

## 初级1 第4课

### 1. 今 ～時 ～分です — 现在几点几分

- 结构：**今(いま)＋数字＋時(じ)＋数字＋分(ふん/ぷん)＋です**
- 「分」的读法规则：

| 分 | 1分 | 2分 | 3分 | 4分 | 5分 | 6分 | 7分 | 8分 | 9分 | 10分 |
|---|---|---|---|---|---|---|---|---|---|---|
| 读音 | いっ**ぷん** | に**ふん** | さん**ぷん** | よん**ぷん** | ご**ふん** | ろっ**ぷん** | なな**ふん** | はっ**ぷん** | きゅう**ふん** | じゅっ**ぷん** |

- ⚠️ 1/3/4/6/8/10分读「ぷん」，2/5/7/9分读「ふん」。
- 例：今 何時(なんじ)ですか。→ 7時(じ)10分(ぷん)です。

### 2. 動詞ます形の四種時態 — ます形四种时态

- 结构：

| 时态 | 形式 | 例（勉強する） |
|------|------|--------------|
| 现在肯定 | Vます | 勉強(べんきょう)し**ます** |
| 现在否定 | Vません | 勉強し**ません** |
| 过去肯定 | Vました | 勉強し**ました** |
| 过去否定 | Vませんでした | 勉強し**ませんでした** |

- 「ます」形表示现在的习惯、真理，或将来的动作。「ました」表示过去。
- 例：毎日(まいにち) 勉強**します**。（每天学习。）
- 例：きのう 勉強**しましたか**。→ はい、勉強しました。/ いいえ、勉強しませんでした。

### 3. 時間に 動詞ます — 用「に」标记具体时间点

- 结构：**時間＋に＋動詞ます**
- 具体的时间点（几点、几月几日、星期几）后面加「に」。
- ⚠️ 以下时间词**不加**「に」：きょう、あした、きのう、まいにち、まいあさ、いま、いつ 等。
- ⚠️ 以下时间词「に」**可加可不加**：～ようび、あさ、ひる、ばん、よる。
- 例：6時半(じはん)**に** 起(お)きます。（6点半起床。）
- 例：**きのう** 勉強しました。（昨天学习了。→ 不加に）

### 4. ～から ～まで — 从...到...

- 结构：**名詞1＋から＋名詞2＋まで**
- 「から」表示起点，「まで」表示终点。可以单独使用。
- 例：9時**から** 5時**まで** 勉強します。（从9点学到5点。）
- 例：銀行(ぎんこう)は 9時**から** 3時**まで**です。（银行从9点到3点营业。）

### 5. 名詞1と 名詞2 — 和（并列）

- 结构：**名詞1＋と＋名詞2**
- 「と」连接名词，表示并列。
- 例：銀行の 休(やす)みは 土曜日(どようび)**と** 日曜日(にちようび)です。（银行的休息日是周六和周日。）

### 6. ～ね — 确认/寻求同意

- 句末加「ね」，表示寻求对方同意或确认。
- 例：毎日 10時まで 勉強します。……大変(たいへん)です**ね**。（每天学到10点。……真辛苦啊。）
- 例：山田さんの 電話番号(でんわばんごう)は 871の6813です**ね**。（山田的电话号码是871-6813对吧。）

## 初级1 第5课

### 1. 場所へ 行きます/来ます/帰ります — 移动方向

- 结构：**場所＋へ＋行(い)きます/来(き)ます/帰(かえ)ります**
- 「へ」表示移动方向，读作「え」。
- 例：京都(きょうと)**へ** 行きます。（去京都。）
- 例：日本(にほん)**へ** 来(き)ました。（来了日本。）
- 例：うち**へ** 帰ります。（回家。）

### 2. どこ[へ]も 行きません — 哪里也不去（全面否定）

- 结构：**疑問詞＋も＋否定動詞**
- 疑问词＋も＋否定 = 全面否定。
- 例：**どこ[へ]も** 行きません。（哪里也不去。）
- 例：**何(なに)も** 食(た)べません。（什么也不吃。）
- 例：**だれも** いません。（谁也不在。）

### 3. 乗り物で 行きます — 交通工具

- 结构：**乗(の)り物(もの)＋で＋移動動詞**
- 「で」标记使用的交通工具。步行用「歩(ある)いて」，不用「で」。
- ⚠️ 步行 =「歩いて」，不说「足で」。
- 例：電車(でんしゃ)**で** 行きます。（坐电车去。）
- 例：タクシー**で** 来ました。（坐出租车来的。）
- 例：駅(えき)から **歩(ある)いて** 帰りました。（从车站走回来的。）

### 4. 人と 動詞ます — 和某人一起做

- 结构：**人＋と＋動詞ます**
- 「と」标记一起做某事的同伴。独自一人用「一人(ひとり)で」。
- 例：家族(かぞく)**と** 日本へ 来ました。（和家人来了日本。）
- 例：**一人で** 東京(とうきょう)へ 行きます。（一个人去东京。）

### 5. いつ — 什么时候

- 「いつ」询问时间，不加「に」。
- 例：**いつ** 日本へ 来ましたか。→ 3月(がつ)25日(にち)に 来ました。
- 例：**いつ** 広島(ひろしま)へ 行きますか。→ 来週(らいしゅう) 行きます。

### 6. ～よ — 提供信息/主张

- 句末加「よ」，向对方提供新信息或表达自己的主张。
- ⚠️ 「ね」是寻求同意（你也这么觉得吧），「よ」是告知对方（我告诉你哦）。
- 例：この 電車(でんしゃ)は 甲子園(こうしえん)へ 行きますか。→ いいえ、次(つぎ)の「普通(ふつう)」です**よ**。

## 初级1 第6课

### 1. 名詞を 動詞ます — 宾语标记

- 结构：**名詞＋を＋他動詞ます**
- 「を」标记动作的对象（宾语），读作「お」。
- 例：ジュース**を** 飲(の)みます。（喝果汁。）
- 例：新聞(しんぶん)**を** 読(よ)みます。（读报纸。）

### 2. 名詞を します — 做某活动

- 结构：**名詞＋を＋します**
- 「します」可以接多种名词，表示"做某事"。
- 适用范围：运动（サッカーをします）、游戏（トランプをします）、活动（パーティーをします）、工作（仕事(しごと)をします）、学习（宿題(しゅくだい)をします）、通讯（電話(でんわ)をします）。
- 例：サッカー**を** します。（踢足球。）
- 例：宿題**を** しました。（做了作业。）

### 3. 何を しますか — 做什么

- 结构：**何(なに)を しますか**
- 询问对方要做什么/做了什么。回答可以是具体动作。
- 例：月曜日(げつようび) **何を** しますか。→ 京都へ 行きます。
- 例：きのう **何を** しましたか。→ サッカーを しました。

### 4. なに vs なん — 「何」的两种读法

- 「何」有两种读法，取决于后续音节：
  - **なん**：后接「た行・だ行・な行」的音时。也用于「何＋量词」。
  - **なに**：其他情况。
- 例：それは **何(なん)**ですか。（那是什么？→ 后接「で」= だ行）
- 例：**何(なに)を** しますか。（做什么？→ 后接「を」= 非た/だ/な行）
- 例：**何(なん)の** 本ですか。（什么书？→ 后接「の」= な行）

### 5. 場所で 動詞ます — 动作发生的场所

- 结构：**場所＋で＋動詞ます**
- 「で」标记动作发生的地点。
- ⚠️ 区分：「へ」= 移动方向，「で」= 动作场所。
- 例：駅(えき)**で** 新聞を 買(か)います。（在车站买报纸。）
- 例：図書館(としょかん)**で** 勉強します。（在图书馆学习。）

### 6. 動詞ませんか — 邀请

- 结构：**動詞＋ませんか**
- 用否定疑问形式表示邀请，比直接说更委婉。
- 例：一緒(いっしょ)に 東京へ 行き**ませんか**。→ ええ、いいですね。（一起去东京好吗？→ 好啊。）

### 7. 動詞ましょう — 提议一起做

- 结构：**動詞＋ましょう**
- 积极提议一起做某事，或接受邀请时使用。
- 例：ちょっと 休(やす)み**ましょう**。（稍微休息一下吧。）
- 例：一緒に 昼(ひる)ご飯(はん)を 食べ**ましょう**。→ ええ、食べましょう。

## 初级1 第7课

### 1. 名詞（道具）で＋動詞 — 用...做...（方式/工具）

- 结构：**名詞＋で＋動詞**
- 「で」标记做某事使用的工具、手段、方法或语言。
- 例：箸(はし)**で** ご飯(はん)を 食(た)べます。（用筷子吃饭。）
- 例：日本語(にほんご)**で** レポートを 書(か)きます。（用日语写报告。）

### 2.「言葉(ことば)」は ～語(ご)で 何(なん)ですか — 用某语言怎么说

- 结构：**「词」は ～語で 何ですか**
- 询问某个词用某种语言怎么说。
- 例：「ありがとう」は 英語(えいご)で 何(なん)ですか。→「Thank you」です。
- 例：「Thank you」は 日本語(にほんご)で 何(なん)ですか。→「ありがとう」です。

### 3. 名詞（人）に 名詞を あげます — 给某人某物

- 结构：**（わたしは）人＋に＋物を＋あげます**
- 「に」标记接受者（给谁）。「あげます」= 给。
- 注意：主语是"我"或"我方"给别人，不能说别人给我用あげます。
- 例：わたしは 木村(きむら)さん**に** 花(はな)を あげます。（我给木村花。）

### 4. 名詞（人）に／から 名詞を もらいます — 从某人那里得到

- 结构：**（わたしは）人＋に／から＋物を＋もらいます**
- 「に」或「から」标记给予者（从谁那里）。「もらいます」= 得到/收到。
- 「から」更强调"来源"，组织/公司等用「から」更自然。
- 例：わたしは カリナさん**に** チョコレートを もらいました。（我从卡莉娜那里收到了巧克力。）

### 5. もう＋動詞ました — 已经做了

- 结构：**もう＋Vました**
- 表示动作已经完成。
- 肯定回答：はい、もう Vました。
- 否定回答：いいえ、**まだです**。（不，还没有。）
- 例：もう メールを 送(おく)りました。（已经发了邮件。）
- 例：もう 昼(ひる)ご飯(はん)を 食(た)べましたか。→ いいえ、まだです。

## 初级1 第8课

### 1. い形容詞 / な形容詞 — 形容词基本用法

- い形容词：词尾是「い」，直接加「です」做谓语，直接修饰名词。
  - 谓语：**暑(あつ)い です**（很热）
  - 修饰名词：**高(たか)い 山(やま)**（高的山）
- な形容词：加「です」做谓语，加「な」修饰名词。
  - 谓语：**静(しず)か です**（安静）
  - 修饰名词：**静(しず)か**な** 町(まち)**（安静的城市）
- ⚠️ 陷阱：「きれい」「有名(ゆうめい)」虽然看起来像い形容词，但它们是**な形容词**。

### 2. い形容詞 / な形容詞の否定形 — 形容词否定（现在）

| 类型 | 肯定 | 否定变形方法 | 否定形 |
|------|------|-------------|--------|
| い形容词 | 暑い です | 去「い」＋**くないです** | 暑**くないです** |
| な形容词 | 静か です | 词干＋**じゃありません** | 静か**じゃありません** |

- ⚠️ 特殊：**いい**（好）→ **よくないです**（不好）。「いい」的所有变形都从「よ」开始。
- 例：この 映画(えいが)は おもしろ**くないです**。（这部电影不有趣。）
- 例：ここは 静(しず)か**じゃありません**。（这里不安静。）

### 3. い形容詞 / な形容詞の過去形 — 形容词过去式

| 类型 | 过去肯定变形 | 过去肯定形 | 过去否定变形 | 过去否定形 |
|------|------------|-----------|------------|-----------|
| い形容词 | 去「い」＋**かったです** | 暑**かったです** | 去「い」＋**くなかったです** | 暑**くなかったです** |
| な形容词 | 词干＋**でした** | 静か**でした** | 词干＋**じゃありませんでした** | 静か**じゃありませんでした** |

- ⚠️ 特殊：**いい** → 过去肯定：**よかったです** / 过去否定：**よくなかったです**
- 例：昨日(きのう)は 暑(あつ)**かったです**。（昨天很热。）
- 例：旅行(りょこう)は 大変(たいへん)**じゃありませんでした**。（旅行不辛苦。）

### 4. ～は どうですか — 询问感想/评价

- 结构：**名詞は どうですか**
- 询问对方对某事物的感想或评价。
- 例：日本(にほん)の 生活(せいかつ)は どうですか。→ 楽(たの)しいです。

### 5. ～は どんな 名詞ですか — 询问性质/特征

- 结构：**名詞1は どんな 名詞2ですか**
- 「どんな」后必须接名词，询问"是什么样的..."。
- 例：奈良(なら)は どんな 町(まち)ですか。→ 静(しず)かな 町(まち)です。

### 6. 名詞1は 名詞2が ～ — 对象提示（好恶/能力）

- 结构：**人は 对象が 好き/嫌い/上手/下手/わかります**
- 「が」标记好恶、能力等的对象。
- 例：わたしは イタリア料理(りょうり)**が** 好(す)きです。（我喜欢意大利菜。）
- 例：わたしは 日本語(にほんご)**が** 少(すこ)し わかります。（我稍微懂日语。）

### 7. ～から — 原因/理由

- 结构：**原因句＋から、结果句** 或 **结果句。原因句＋から。**
- 表示原因或理由。
- 例：時間(じかん)が ありません**から**、新聞(しんぶん)を 読(よ)みません。（因为没时间，所以不看报纸。）
- 如果句子太长，可以拆开：新聞を 読みません。時間が ありませんから。

### 1. 名詞が あります — 拥有某物

- 结构：**（人は）名詞が あります**
- 表示拥有某物。「が」标记拥有的对象。
- 例：わたしは 車(くるま)**が** あります。（我有车。）
- 例：時間(じかん)**が** ありますか。→ いいえ、ありません。

### 2. 名詞が 好き/嫌い/上手/下手 です — 好恶/擅长

- 结构：**人は 名詞が 好き/嫌い/上手/下手 です**
- 对象用「が」标记。程度可加「とても」「あまり（＋否定）」等。
- 好き（喜欢）↔ 嫌い（讨厌）、上手（擅长）↔ 下手（不擅长）

### 3. 名詞が わかります — 理解/懂得

- 结构：**人は 名詞が わかります**

### 4. 名詞が 欲(ほ)しいです — 想要某物

- 结构：**（わたしは）名詞が 欲しいです**
- 只用于第一人称（我想要）。对象用「が」。
- 否定：欲しくないです。
- 例：わたしは 新(あたら)しい 車(くるま)**が** 欲(ほ)しいです。（我想要新车。）

### 5. 動詞ます形（去ます）＋たいです — 想做某事

- 结构：**Vます形去掉「ます」＋たいです**
- 只用于第一人称。变形：行きます → 行き**たいです**、食べます → 食べ**たいです**
- 否定：～たくないです。

### 6. 名詞/Vます形（去ます）＋に 行きます/来ます/帰ります — 移动的目的

- 结构：**場所へ＋Vます形去ます＋に＋行きます/来ます/帰ります**
- 或：**場所へ＋名詞＋に＋行きます/来ます/帰ります**
- 「に」前面接目的（做什么去/来）。
- 例：神戸(こうべ)へ インド料理(りょうり)を 食(た)べ**に** 行(い)きます。（去神户吃印度菜。）

- 结构：**疑问词＋か** = 某处/某物/某人
- どこか（某处）、何か（某物）、だれか（某人）
- 例：冬休(ふゆやす)みは **どこか** 行(い)きましたか。（寒假去了什么地方吗？）
- 例：**何(なに)か** 飲(の)みたいです。（想喝点什么。）

## 初级1 第14课

### 1. 動詞のグループ分類 — 动词分三组

- 判断方法：看「ます形」去掉「ます」后的最后一个音。
- **I类（1グループ）**：去ます后最后一个音在「い段」。
  - 例：書(か)**き**ます、飲(の)**み**ます、買(か)**い**ます、話(はな)**し**ます
- **II类（2グループ）**：去ます后最后一个音在「え段」，或以下い段特殊动词：
  - 特殊II类：います、起(お)きます、借(か)ります、着(き)ます、見(み)ます、浴(あ)びます、信(しん)じます、できます、降(お)ります、落(お)ちます
  - 例：食(た)**べ**ます、見(み)**せ**ます、寝(ね)ます
- **III类（3グループ）**：只有两个系列：
  - します（做）、来(き)ます（来）
  - 2个字以上汉字/外来语＋します → III类（勉強します、コピーします）
  - ⚠️ 1个汉字＋します → I类（貸(か)します、話(はな)します）

### 2. 動詞て形の作り方 — て形变形规则

- **I类**：根据ます前的词尾音变化（音便）：

| ます前词尾 | て形变化 | 例 |
|-----------|---------|-----|
| い、ち、り | → **って** | 買います→買**って**、待ちます→待**って**、帰ります→帰**って** |
| み、び、に | → **んで** | 飲みます→飲**んで**、遊びます→遊**んで**、死にます→死**んで** |
| き | → **いて** | 書きます→書**いて** ⚠️例外：行きます→行**って** |
| ぎ | → **いで** | 泳ぎます→泳**いで** |
| し | → **して** | 話します→話**して** |

- **II类**：去掉「ます」＋**て**。例：食べます→食べ**て**、見ます→見**て**
- **III类**：します→**して**、来(き)ます→来(き)**て**

### 3. Vて形＋ください — 请求（请做...）

- 结构：**Vて形＋ください**
- 请求对方做某事。口语中可省略「ください」。
- 例：ちょっと 待(ま)って **ください**。（请稍等。）
- 例：すみませんが、名前(なまえ)を 書(か)いて ください。（不好意思，请写名字。）

### 4. Vて形＋います — 正在进行

- 结构：**Vて形＋います**
- 表示动作正在进行中。
- 例：ミラーさんは 今(いま) 電話(でんわ)を かけて **います**。（米勒先生现在正在打电话。）
- 例：雨(あめ)が 降(ふ)って **います**。（正在下雨。）

- 结构：**Vて形＋も いいですか**
- 请求许可，"可以做...吗？"
- 肯定回答：はい、いいですよ。/ はい、どうぞ。
- 否定回答：すみません、ちょっと…。
- 例：写真(しゃしん)を 撮(と)って**も いいですか**。（可以拍照吗？）

### 2. Vて形＋は いけません — 禁止

- 例：ここで タバコを 吸(す)って**は いけません**。（这里不可以吸烟。）

### 3. Vて形＋います — 习惯/结果状态/职业（扩展）

- 结构：**Vて形＋います**
- 除了"正在进行"外，还有三种重要用法：
- **① 结果状态的持续**：动作已完成，结果状态持续。
  - 常见状态动词：住みます、結婚します、持ちます、知ります、かけます（戴眼镜）
  - 例：わたしは 結婚(けっこん)して **います**。（我结婚了。→ 已婚状态）

  - 例：マリアさんは どこに 住(す)んで **いますか**。（玛丽亚住在哪里？）
- **② 习惯性动作**：长期重复做某事。
  - 例：毎朝(まいあさ) ジョギングを して **います**。（每天早上跑步。）
- **③ 职业**：表示从事某种工作。
  - 例：IMCで 働(はたら)いて **います**。（在IMC工作。）
- ⚠️ 「知ります」的否定：**知りません**（不是「知っていません」）

- 用て形按时间顺序连接多个动作。
- 例：朝(あさ) ジョギングを **して**、シャワーを **浴(あ)びて**、会社(かいしゃ)へ 行(い)きます。（早上跑步，洗澡，去公司。）

### 1. Vて形の接続 — 动作顺序连接（复习）

- 结构：**V1て、V2て、V3ます**
- 用て形按时间顺序连接多个动作。

- 表示先做A再做B。强调A完成后才做B。与单纯て形连接不同，「てから」更强调先后顺序。

- **い形容词**：去「い」＋**くて**

  - 例：奈良(なら)は 静(しず)か**で**、きれいな 町(まち)です。（奈良是安静又漂亮的城市。）

## 初级1 第18课

### 1. 動詞辞書形の作り方 — 辞书形（原形）变形规则

- 辞书形 = 动词的原形/字典形，是ます形的普通体现在肯定形。
- **I类**：ます前的「い段」音 → 变为「う段」音

| ます前 | → う段 | 例 |
|--------|-------|-----|
| き(ki) | → く(ku) | 書きます→書**く** |
| み(mi) | → む(mu) | 飲みます→飲**む** |
| り(ri) | → る(ru) | 帰ります→帰**る** |
| し(shi) | → す(su) | 話します→話**す** |
| い(i) | → う(u) | 買います→買**う** |

- **II类**：去掉「ます」＋**る**。例：食べます→食べ**る**、見ます→見**る**
- **III类**：します→**する**、来(き)ます→来(く)**る**

### 2. 名詞が できます — 能力/可能

- 结构：**人は 名詞が できます**
- 表示有某种能力或技能。对象用「が」。
- 例：ミラーさんは 漢字(かんじ)**が** できます。（米勒先生会汉字。）
- 例：ギター**が** できますか。（会弹吉他吗？）

### 3. V辞書形＋ことが できます — 能做某事

- 结构：**V辞書形＋ことが できます**
- 「こと」将动词名词化。表示有能力做某事或制度上允许做某事。
- 例：日本語(にほんご)で 手紙(てがみ)を 書(か)**く ことが できます**。（能用日语写信。）
- 例：カードで 払(はら)**う ことが できます**か。（可以用卡支付吗？）
- 例：運転中(うんてんちゅう)、勝手(かって)に バスを 降(お)り**る ことが できません**。（驾驶中不能随便下车。）

### 4. 趣味は V辞書形＋ことです — 兴趣爱好

- 结构：**趣味は V辞書形＋ことです**
- 用辞书形＋こと 将动词名词化，表示兴趣。
- 例：わたしの 趣味(しゅみ)は 映画(えいが)を 見(み)**る ことです**。（我的兴趣是看电影。）
- 例：わたしの 趣味(しゅみ)は 音楽(おんがく)を 聴(き)**く ことです**。（我的爱好是听音乐。）

### 5. V辞書形/名詞の＋まえに — 在...之前

- 结构：**V辞書形＋まえに、V2** 或 **名詞の＋まえに、V2**
- 表示在做某事之前。动词用辞书形，名词用「の」连接。
- 例：寝(ね)**る まえに**、日記(にっき)を 書(か)きます。（睡觉之前写日记。）
- 例：食事(しょくじ)**の まえに**、手(て)を 洗(あら)います。（吃饭之前洗手。）

### 6. 疑問詞＋でも — 全面肯定（任何...都...）

- 结构：**疑问词＋でも**
- いつでも（任何时候）、だれでも（任何人）、どこでも（任何地方）、何(なん)でも（任何东西）
- 例：**いつでも** **だれでも** この 銀行(ぎんこう)に お金(かね)を 借(か)りる ことが できます。（任何时候任何人都可以在这家银行借钱。）

### 7. 何回(なんかい)も＋V — 好多次

- 结构：**何回も＋動詞**
- 表示多次重复。
- 例：**何回(なんかい)も** 日本(にほん)へ 行(い)った ことが あります。（去过日本好多次。）

## 初级1 第19课

### 1. 動詞た形の作り方 — た形变形规则

- た形 = て形把「て/で」换成「た/だ」。变形规则与て形完全相同。
- **I类**：

| ます前词尾 | た形变化 | 例 |
|-----------|---------|-----|
| い、ち、り | → **った** | 買います→買**った**、待ちます→待**った**、帰ります→帰**った** |
| み、び、に | → **んだ** | 飲みます→飲**んだ**、遊びます→遊**んだ** |
| き | → **いた** | 書きます→書**いた** ⚠️例外：行きます→行**った** |
| ぎ | → **いだ** | 泳ぎます→泳**いだ** |
| し | → **した** | 話します→話**した** |

- **II类**：去掉「ます」＋**た**。例：食べます→食べ**た**、見ます→見**た**
- **III类**：します→**した**、来(き)ます→来(き)**た**

### 2. Vた形＋ことが あります — 经验（曾经做过...）

- 结构：**Vた形＋ことが あります**
- 表示曾经有过某种经历。否定用「ことが ありません」。
- 例：馬(うま)に 乗(の)**った ことが あります**。（骑过马。）
- 例：北海道(ほっかいどう)へ 行(い)**った ことが ありません**。（没去过北海道。）
- 例：日本(にほん)の 映画(えいが)を 見(み)**た ことが ありますか**。（看过日本电影吗？）

### 3. Vた形＋り、Vた形＋り します — 列举动作

- 结构：**V1た形＋り、V2た形＋り します**
- 列举若干有代表性的动作，暗示还有其他（...啦...啦之类的）。
- 句末的「します」要根据时态变化（します/しました）。
- 例：日曜日(にちようび)は 買(か)い物(もの)し**たり**、映画(えいが)を 見(み)**たり** します。（星期天买买东西、看看电影什么的。）
- 例：休(やす)みの 日(ひ)は テニスを し**たり**、散歩(さんぽ)に 行(い)っ**たり** します。（休息日打打网球、散散步什么的。）

### 4. い形容詞（～く）/ な形容詞（～に）＋なります — 变化（变得...）

- 结构：
  - **い形容词**：去「い」＋**く なります**
  - **な形容词**：词干＋**に なります**
  - **名词**：名詞＋**に なります**
- 表示状态变化，"变得..."。
- 例：日本語(にほんご)が 上手(じょうず)**に** なりました。（日语变好了。）
- 例：暖(あたた)か**く** なりました。（变暖和了。）
- 例：だんだん 寒(さむ)**く** なります。（渐渐变冷。）
- 例：朝(あさ)は いい 天気(てんき)でしたが、午後(ごご)から 雨(あめ)**に** なりました。（早上天气好，但下午变成了雨天。）

## 初级1 第20课（截止0212）

### 1. 普通形（普通体/简体）— 完整对照表

- 与丁宁体（ます/です）相对的口语体。用于朋友间对话、从句中。

**动词的普通形：**

| 时态 | 丁寧体 | → | 普通体 | 变形名称 |
|------|--------|---|--------|----------|
| 肯定（非过去） | 行きます | → | 行く | 辞書形 |
| 否定（非过去） | 行きません | → | 行かない | ない形 |
| 肯定（过去） | 行きました | → | 行った | た形 |
| 否定（过去） | 行きませんでした | → | 行かなかった | ない形去い＋かった |

- 过去否定构成：ない形去掉「い」＋**かった**（行かな~~い~~ → 行かな**かった**）

**い形容词的普通形：**

| 时态 | 丁寧体 | → | 普通体 | 变形方法 |
|------|--------|---|--------|----------|
| 肯定（非过去） | 暑い です | → | 暑い | い です→**い** |
| 否定（非过去） | 暑くない です | → | 暑くない | くないです→**くない** |
| 肯定（过去） | 暑かった です | → | 暑かった | かったです→**かった** |
| 否定（过去） | 暑くなかった です | → | 暑くなかった | くなかったです→**くなかった** |

- 要点：い形容词的普通形 = **去掉「です」**即可。
- ⚠️ 特殊：**いい** → よくない / よかった / よくなかった

**な形容词的普通形：**

| 时态 | 丁寧体 | → | 普通体 | 变形方法 |
|------|--------|---|--------|----------|
| 肯定（非过去） | 静か です | → | 静か**だ** | です→**だ** |
| 否定（非过去） | 静か じゃありません | → | 静か**じゃない** | じゃありません→**じゃない** |
| 肯定（过去） | 静か でした | → | 静か**だった** | でした→**だった** |
| 否定（过去） | 静か じゃありませんでした | → | 静か**じゃなかった** | じゃありませんでした→**じゃなかった** |

**名词文的普通形：** 同な形容词

| 时态 | 丁寧体 | → | 普通体 | 变形方法 |
|------|--------|---|--------|----------|
| 肯定（非过去） | 雨 です | → | 雨**だ** | 同な形容詞 |
| 否定（非过去） | 雨 じゃありません | → | 雨**じゃない** | 同な形容詞 |
| 肯定（过去） | 雨 でした | → | 雨**だった** | 同な形容詞 |
| 否定（过去） | 雨 じゃありませんでした | → | 雨**じゃなかった** | 同な形容詞 |

### 2. 普通形の使い方 — 简体的使用场景

- 亲密关系间的日常对话使用简体。
- **提问**：升调（↑），不加「か」。**回答**：うん（嗯）/ ううん（不是）。
- 男性句末常加「だ」「だよ」，女性较少用「だ」，常用「よ」。
- 例：明日(あした) 行(い)く？（明天去吗？）→ うん、行(い)く。（嗯，去。）
- 例：この 映画(えいが)、おもしろかった？（这电影有趣吗？）
- 例：あの 人(ひと)、だれ？→ サントスさん**だよ**。（男性）/ サントスさん**よ**。（女性）
- **从句中使用**：从句内部必须用普通形。
  - 例：あした 雨が 降(ふ)**る**と 思(おも)います。（我想明天会下雨。）

## 初级1 第21课

### 1. 普通形 + と思います — 我觉得…/我认为…

- 结构：**普通形 + と思います**
- 表达说话人的想法、意见、推测。「と」是引用助词。

| 词类 | 接续 |
|------|------|
| 动词 | 行く + と思います |
| い形 | おいしい + と思います |
| な形 | 静か**だ** + と思います |
| 名词 | 雨**だ** + と思います |

- 例：あした雨が降ると思います。（我觉得明天会下雨。）
- 例：日本はきれいだと思います。（我觉得日本很漂亮。）
- ⚠️ な形容词・名词的现在肯定要加「だ」再接「と思います」

### 2. 普通形 + と言います — 某人说…

- 结构：**普通形 + と言います/言いました**
- 转述别人说的话（间接引语）。
- 例：ミラーさんは来週東京へ行くと言いました。（米勒说下周去东京。）

### 3. 「～」と言います — 直接引用

- 结构：**「原话」+ と言います**
- 直接引用原话，用「」括起来。也用于解释词语。
- 例：「ありがとう」は中国語で「谢谢」と言います。

### 4. 普通形 + でしょう？ — …吧？（征求确认）

- 结构：**普通形 + でしょう？（升调）**
- 向对方确认自己认为正确的信息，期待对方同意。
- ⚠️ な形容词・名词：**去掉「だ」** 直接加でしょう？

| 词类 | 接续 |
|------|------|
| 动词 | 来る + でしょう？ |
| い形 | おいしい + でしょう？ |
| な形 | 静か~~だ~~ + でしょう？ |
| 名词 | 日曜日~~だ~~ + でしょう？ |

- 例：明日は日曜日でしょう？（明天是星期天吧？）
- 例：このケーキ、おいしいでしょう？（这个蛋糕好吃吧？）

### 5. 場所 + で + 事件/活動 + があります — 在某地举行/发生…

- 结构：**場所 + で + 事件 + があります**
- ⚠️ 与第10课「場所**に**Nがあります」（物体存在）区分：事件/活动用「**で**」
- 例：東京でオリンピックがありました。（在东京举办了奥运会。）
- 例：7月25日に大阪で天神祭りがあります。（7月25日在大阪有天神祭。）

### 6. 期間 + に + 回数 + 動詞 — 在某段时间内做几次

- 结构：**期間 + に + 回数 + 動詞**
- 「に」表示基准时间段。

| 次数 | 读法 |
|------|------|
| 1回 | いっかい |
| 2回 | にかい |
| 3回 | さんかい |

- 例：1か月に2回映画を見ます。（一个月看两次电影。）
- 例：1日に3回薬を飲みます。（一天吃三次药。）

### 7. 名詞 + でも — …什么的/…之类的

- 结构：**名詞 + でも + 動詞ませんか**
- 举出代表性例子作为提议。
- 例：ビールでも飲みませんか。（喝杯啤酒什么的？）

## 初级1 第22课

### 1. 連体修飾（名詞修飾）— 用普通形句子修饰名词

- 结构：**普通形（简体句） + 名詞**
- 用动词/形容词/名词的普通形放在名词前修饰，相当于"……的（名词）"。

| 词类 | 现在肯定 | 现在否定 |
|------|---------|---------|
| 动词 | 読む＋名詞 | 読まない＋名詞 |
| い形 | おいしい＋名詞 | おいしくない＋名詞 |
| な形 | 静か**な**＋名詞 | 静かじゃない＋名詞 |
| 名词 | 猫**の**＋名詞 | 猫じゃない＋名詞 |

- ⚠️ な形容词现在肯定：用「**な**」不用「だ」（静か**な**部屋）
- ⚠️ 名词现在肯定：用「**の**」不用「だ」（猫**の**写真）
- ⚠️ 修饰句中的主语用「**が**」不用「は」

- 例：これは私がいつも読んでいる雑誌です。（这是我经常看的杂志。）
- 例：きのう習ったことばを忘れました。（忘了昨天学的单词。）
- 例：日本語を使う仕事をしたいです。（想做使用日语的工作。）

### 2. どんな＋名詞 — 什么样的…

- 结构：**どんな + 名詞**
- 询问性质/特征，用修饰句来回答。
- 例：どんな人と結婚したいですか。（想和什么样的人结婚？）
- 例：友達が多くて、ユーモアがある人と結婚したいです。（想和朋友多、有幽默感的人结婚。）
